# PDFs Directory

This directory should contain the following PDF files:

1. **Staff-Guide-En-Fr.pdf** - Staff training guide in English and French
2. **BrochureEN-FR-ZH.pdf** - Product brochure in English, French, and Chinese
3. **Christmas-POS-EN.pdf** - Christmas point-of-sale promotional materials

## To Add PDFs:

Upload these files from your Google Drive or local storage to this directory.

The index.html file is already configured to link to these PDFs.
